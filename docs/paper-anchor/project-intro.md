# 概述

## 纸锚 (Paper Anchor)

> **让思考轻盈如纸，让证据稳固如锚。**

### 项目介绍

**纸锚**是一个采用“专注写作，独立存储”理念构建的下一代智能 AI 知识库。

在信息过载的时代，我们将创作过程重新拆解为两个互不干扰却逻辑深连的维度：**“纸”**（笔记空间）与**“锚”**（资源网盘）。系统摒弃了传统笔记软件将文档与素材混杂的管理方式，通过独创的双工作台架构，让您在极致纯净的环境中捕捉灵感，同时利用 AI 技术静默处理海量参考资料。

无论是一闪而过的灵感，还是厚重的行业报告，在纸锚中都能找到各自的归宿，并通过隐形的逻辑链路完美融合。

### 核心特性

#### 1. 纸：极致纯净的写作空间 (Note Space)
位于 `/dashboard/notes` 的笔记空间是思考的容器。
* **极简主义设计**：摒弃一切干扰元素，提供全屏宽度的 Tiptap 编辑器，专注于 Markdown 文本创作。
* **流式思维**：通过双栏布局快速切换笔记，支持按时间或标题排序，捕捉稍纵即逝的想法。

#### 2. 锚：自动化的资源深井 (Resource Drive)
位于 `/dashboard/resources` 的资源网盘是知识的压舱石。
* **静默 AI 处理**：将 PDF 等重资产拖入网盘后，后台 Edge Function 会自动运行，静默完成文件解析、切片与向量化存储 (Embedding)。
* **状态可视化**：清晰展示文件从“等待中”到“已索引”的全过程，让数据处理进度一目了然 。

#### 3. 链：事实与观点的强关联
通过引用语法将“纸”与“锚”无缝连接 [cite: 4]。
* **引用写作**：在写作时只需输入 `[[`，即可唤起资源搜索框，将后台已索引的行业报告或文献一键插入文档 。
* **溯源阅读**：生成的胶囊链接 `<ResourceLink />` 是连接思考与事实的锚点，点击即可在预览界面查阅原始文件，确保每一行文字都有据可依 。

### 技术架构 (v2.3)
* **双核数据库**：采用解耦设计，`resources` 表管理物理资产，`knowledge_nodes` 表管理笔记与 AI 切片数据，二者通过 `node_links` 实现逻辑打通。
* **智能 RAG 引擎**：非结构化文件上传即刻转为向量数据 (`embedding vector`)，为未来的语义搜索与 AI 问答奠定基础。
