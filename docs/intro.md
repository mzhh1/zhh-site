# 概述

这里是纸灰灰有趣的项目！

项目有不同的状态。包括`构思`，`推进`，`维护`和`停止`。

## 星枢沙盒

> 和AI或者朋友来一局狼人杀吧!

`维护` `应用`

人与AI共同游戏的在线回合制游戏平台

游玩方式支持**人vs人**，**人vsAI**，**AI vs AI**

游戏支持**井字棋**，**五子棋**，**象棋**，**狼人杀**

推荐在**狼人杀**尝试*gpt4.1*,表现极佳。**五子棋**，**象棋**等强推理游戏AI能力较弱。
