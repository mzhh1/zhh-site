# 概述

# **星枢沙盒 (NexusPlayground)**

**AI回合制游戏平台**

### **基础信息**

“纸核”是一个通用的 LLM 应用开发平台。它既是开发者的**生产力基座**，也是终端用户的**AI 万能通行证**。

我们旨在帮助团队以极低成本、在可控合规的边界内，快速构建具备身份认证、数据治理、积分计费与 LLM 调用能力的业务应用。

* **项目网址:** [https://114.132.91.247/](https://114.132.91.247/)
* **GitHub 仓库:** [github.com/mzhh1/autolab\_auth](https://github.com/mzhh1/autolab_auth)  
* **NPM SDK:** [@autolabz/*](https://www.npmjs.com/settings/autolabz/packages)

### **双向定位**

我们同时服务于应用的使用者与构建者，连接 AI 服务的“最后一公里”。

#### **👤 面向用户：你的 AI 统一通行证**

纸核是您进入 AI 应用生态的统一身份。

* **一键登录：** 无需在不同应用间重复注册，一个纸核账号即可安全登录所有接入应用（如 DeepResearch, informedWU 等）。  
* **资产通用：** 您的积分余额在所有应用间共享，实现跨应用的统一结算与消费。  
* **数据互联：** 在您的授权下，构建应用间的数据桥梁，享受无缝衔接的 AI 服务体验。

#### **🛠️ 面向开发者：你的全栈 LLM 后端**

纸核为您沉淀了构建 LLM 应用所需的一切通用能力。告别重复造轮子，专注于核心业务逻辑。

* **标准身份认证：** 提供开箱即用的 OAuth 2.0 方案（支持 PKCE），无论是内部门户还是第三方集成，都能快速接入。  
* **LLM 自动计费：** 这是我们的核心优势。开发者只需调用 LLM 接口，纸核将自动完成\*\*“积分校验 \-\> 流式响应 \-\> 自动按量扣费 \-\> 生成账单明细”\*\*的全闭环，极大降低了应用商业化的门槛。  
* **统一数据治理：** 通过平台网关与 SDK 统一注入鉴权，实现细粒度的数据访问控制，确保应用合规。  
* **积分资源体系：** 提供通用的用户账户资源系统，让您的应用天生具备成本可视与用量结算能力。

### **核心价值**

对开发者而言，纸核意味着更快的交付（Fast Delivery）与更低的成本（Low Cost）；  
对用户而言，纸核意味着更一致的体验（Consistent UX）与更互联的生态（Connected Ecosystem）。
